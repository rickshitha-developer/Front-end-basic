CSS Basics

CSS (Cascading Style Sheets) brings your web pages to life — colors, layouts, animations, and full responsiveness!

What Can CSS Do?

  • Change colors, fonts, and spacing
  • Create responsive layouts with Flexbox/Grid
  • Add transitions, animations, hover effects
  • Build interactive pages with media queries

Types of CSS

TypeDescriptionExample
Inline Inside HTML tag <p style="color:red;">Text</p>
Internal Inside <style> tag <style>p { color: red; }</style>
External Linked .css file <link rel="stylesheet" href="style.css">

Fonts & Text Effects

  • font-family: Choose fonts
  • font-size: px, em, rem
  • font-weight: Boldness
  • text-align: alignment
  • text-decoration: underline, none
  • text-transform: uppercase, lowercase
  • line-height: spacing between lines
  • letter-spacing: spacing between letters
  • text-shadow: text shadow
.heading {
  font-family: 'Arial', sans-serif;
  text-transform: uppercase;
  text-shadow: 2px 2px #888;
}

Borders & Box Shadows

  • border: style, width, color
  • border-radius: round corners
  • box-shadow: add depth
.card {
  border: 2px solid #000;
  border-radius: 10px;
  box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

Transform & Transition

  • transform: rotate, scale, translate
  • transition: smooth change
.btn {
  transition: transform 0.3s;
}

.btn:hover {
  transform: scale(1.1);
}

Display & Visibility

  • display: block | inline | inline-block | flex | grid | none
  • visibility: visible | hidden
.hidden { display: none; }

CSS Selectors

Selectors target HTML elements to style them:

  • Element: h1 { color: green; }
  • Class: .btn { padding: 10px; }
  • ID: #header { background: black; }
  • Attribute: a[target="_blank"] { }
  • Pseudo-class: li:first-child
  • Pseudo-element: p::after

Understanding Specificity

When multiple rules target the same element, specificity decides which rule wins:

  • Inline style > ID selector > Class selector > Element selector
  • Use fewer !important overrides — use clean hierarchy instead.
/* Example */
#box { color: red; } /* ID wins */
.box { color: blue; }
div { color: green; }

CSS Units

  • px – Absolute pixel
  • % – Relative to parent
  • em / rem – Relative to font-size
  • vh / vw – Viewport height/width

CSS Variables

Reuse values easily with custom properties:

:root {
  --main-color: #3498db;
}

h1 {
  color: var(--main-color);
}

Position & Z-Index

Control where elements appear on the page:

  • position: static; (default)
  • position: relative; (shift from normal position)
  • position: absolute; (relative to nearest positioned ancestor)
  • position: fixed; (relative to viewport)
  • position: sticky; (sticks on scroll)

z-index controls stacking order (higher z-index comes in front).

Pseudo Classes

They allow you to style elements based on state:

  • :hover – when mouse hovers
  • :active – when clicked
  • :focus – when input is focused
  • :first-child – first element among siblings
a:hover { color: red; }

Media Queries

Make your site responsive across devices:

@media (max-width: 600px) {
  body { background-color: lightblue; }
}

Transitions vs Animations

  • Transition: for simple effects (hover, color)
  • Animation: for continuous movement
/* Transition */
.box:hover { background-color: red; transition: 0.5s; }

/* Animation */
@keyframes slide {
  from { transform: translateX(-100px); }
  to { transform: translateX(0); }
}
