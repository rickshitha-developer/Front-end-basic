Basic Flask App Example 

from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flask!"

if __name__ == "__main__":
    app.run(debug=True)

This simple app defines a single route (/) that shows Hello, Flask!. When you visit localhost:5000, Flask calls the home function and sends back the result.

Routes & Templates

A route tells Flask which function to run for a specific URL. Templates use placeholders to insert Python data into HTML.

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def index():
    return render_template("index.html")

The render_template function finds index.html inside the templates folder and sends it to the browser.

<!-- templates/index.html -->

Handling Forms

Flask handles form data using request. You can accept both GET and POST methods to process user input.

from flask import Flask, request

app = Flask(__name__)

@app.route("/login", methods=["GET", "POST"])
def login():
    if request.method == "POST":
        username = request.form["username"]
        return f"Welcome, {username}!"
    return "<form method='post'><input name='username'><input type='submit'></form>"

This app shows a simple login form and greets the user when they submit their name.
