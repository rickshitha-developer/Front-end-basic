HTML Basics

HTML stands for HyperText Markup Language. It forms the backbone of every web page. With HTML, you define structure, organize content, and integrate media.

What Can HTML Do?

  • Create and structure content using semantic elements
  • Embed images, videos, audio, and external content
  • Form the foundation for styling (CSS) and interaction (JavaScript)

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Output:

Welcome

This is a paragraph.

Semantic vs Non-Semantic

Semantic Tags
  • <header> ‚Äì Defines the top section of a page or section. Usually contains logos, titles, or navigation.
  • <nav> ‚Äì Represents a navigation menu (like links to other pages or sections).
  • <main> ‚Äì Represents the main content of a document. Only one <main> should be used per page.
  • <section> ‚Äì Groups related content together. Often used with headings to define different topics.
  • <article> ‚Äì Represents independent, self-contained content like a blog post, news article, or forum post.
  • <footer> ‚Äì Defines the bottom section of a page or section. Usually includes copyright, contact info, or links.
Non-Semantic Tags
  • <div> ‚Äì A generic block-level container. It has no meaning by itself and is often used for grouping elements for styling or layout.
  • <span> ‚Äì A generic inline container. Used to apply styles or target small pieces of content without affecting layout.
