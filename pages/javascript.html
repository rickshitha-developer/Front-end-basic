JavaScript Basics

JavaScript is the scripting language that brings interactivity to websites. It handles user actions, animations, dynamic content, and API calls.

What Can JavaScript Do?

  • Manipulate HTML & CSS (change content, styles, classes)
  • Handle user events (clicks, input, form submission)
  • Fetch data from servers (AJAX, APIs)
  • Create animations & dynamic effects

Basic Syntax 

<script>
  console.log('Hello World');
</script>

Use <script> tags to write or link JavaScript in your HTML.

Conditional Statements

Conditions help you run code only if something is true or false:

  • if â€“ Runs a block if condition is true.
  • else â€“ Runs if the condition is false.
  • else if â€“ Checks another condition if the first is false.
  • switch â€“ Runs different blocks based on a matching value.

Control your code flow with conditions:

  console.log("Passed!");
} else {
  console.log("Try again!");
}
  
let grade = "B";
switch (grade) {
  case "A":
    console.log("Excellent!");
    break;
  case "B":
    console.log("Good Job!");
    break;
  default:
    console.log("Keep Working!");
}

Functions

Functions save reusable blocks of code. Call them whenever you need them.

Functions perform actions when called:

function greet() {
  alert("Hello!");
}

greet();

Arrow Functions

Arrow functions are a shorter way to write functions: 

// Regular function
function add(x, y) {
  return x + y;
}

// Arrow function
const add = (x, y) => x + y;

console.log(add(2, 3)); // 5

Array Methods

JavaScript arrays come with powerful methods: 

let numbers = [1, 2, 3, 4, 5];

// forEach
numbers.forEach(num => console.log(num));

// map
let squares = numbers.map(num => num * num);
console.log(squares); // [1,4,9,16,25]

// filter
let even = numbers.filter(num => num % 2 === 0);
console.log(even); // [2,4]

Loops

Loops repeat actions automatically:

  • for â€“ Repeat a block a specific number of times.
  • while â€“ Repeat while a condition is true.
  • do...while â€“ Like while but runs at least once.

Loops repeat actions:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

While Loop

let count = 1;
while (count <= 5) {
  console.log(count);
  count++;
}

Do While Loop

let num = 1;
do {
  console.log(num);
  num++;
} while (num <= 3);
